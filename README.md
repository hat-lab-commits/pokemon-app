# ポケモン図鑑アプリ
PokeAPI を使用して作成したポケモン図鑑アプリです。
ポケモン名（日本語）で検索すると、図鑑情報・進化系統・フォーム違いを表示します。

## 機能
ー　ポケモン検索
ー　日本語名表示
ー　タイプ表示
ー　お気に入り登録（localStrage保存)
ー  お気に入り削除機能
ー　日本語名でポケモン検索
ー　公式画像表示（全フォーム対応）
ー　進化ツリーを再帰処理で取得
ー　タイプを日本語＋色付き表示
ー　図鑑説明・分類・捕獲率・タマゴグループ表示
ー　Enterキーで検索可能
ー　レスポンシブを意識したレイアウト
ー　鳴き声機能

## 使用技術

ー  CSS
ー  JavaScript
ー  PokéAPI
ー  localStrage
