# ポケモン図鑑アプリ
ポケモンAPIを使用したwebアプリです。

## 機能
ー　ポケモン検索
ー　日本語名表示
ー　タイプ表示
ー　お気に入り登録（localStrage保存)
ー  お気に入り削除機能

## 使用技術
ー　HTML
ー  CSS
ー  JavaScript
ー  PokéAPI
ー  localStrage

## 学習ポイント
ー　fetchを使ったAPI通信
ー　非同期処理（async / await）
ー  CORSの理解
ー　JSONデータの取り扱い
ー　Git / GitHubによるバージョン管理
